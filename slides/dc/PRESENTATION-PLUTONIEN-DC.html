Chiffrage

Deux formules : projet + maintenance

TJM aligné sur le forfait projet (~700 € / j). La maintenance, ce n’est pas le prix d’un fix : c’est un volume engagé (infra, correctifs, petits ajustements). Les features restent en régie.

Option A

Projet + demi-journée / mois

14 000 € + 350 € / mois
  • Projet complet · passation offerte
  • 0,5 j / mois engagé (~700 € / j)
  • Infra, mises à jour services, bugs
  • Petits ajustements dans le volume
Option B

Projet réduit + 1 journée / mois

12 000 € + 700 € / mois
  • −2 000 € sur le projet
  • 1 j / mois engagé (bugs + mods simples)
  • Plus de marge pour le run au quotidien
  • Features toujours en régie

Repère marché : diagnostic Flowlab 16 200 € · freelance ~18 k€ · DSI ~90 k€/an. Features hors forfait = régie. Détail maintenance en annexe.

Maintenance

Ce que couvre l’engagement mensuel

Volume engagé (0,5 j ou 1 j) : pas le prix d’une modification isolée. Mois calme = dispo / assurance. Dépassement = régie après accord.

Inclus

Infra, correctifs, petits ajustements

Suivi prod, mises à jour services si besoin, sécurité, bugs sur le périmètre livré, mods simples (libellé, règle mineure) dans le volume.

Hors forfait

Features & évolutions métier

Nouvelle fonctionnalité, changement de process majeur, lots hors roadmap : devis ou régie (~700 € / j).

Hors forfait · offert

Bug de recette

Aurait dû être vu en phase beta / go-live : corrigé sans consommer le volume mensuel (garantie projet).

Inclus

Bug post-stabilisation

Sur le périmètre livré, après go-live : pris dans le volume engagé du mois.

Régie

Besoin de feature

Évolution produit ou nouveau lot : chiffrage dédié, hors engagement maintenance.
